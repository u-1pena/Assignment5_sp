## URLについて

URLとは __「Uniform Resource Locator」__ の略称で、インターネット上に存在する情報資源（文章や画像など）の場所を指し示す技術方式。  
簡単に言えばインターネットにおける住所です。アドレスとも呼ばれます。  
通常 __<font color ="green">「プロトコル://ドメイン名/ディレクトリパス名/ファイル名」</font>__ という形式で構成されます。
